# Oh My PM — Codex Pack

Install Oh My PM for Codex-compatible agents.

## What this pack contains

- `AGENTS.md` — Core agent instructions.
- `.agents/skills/oh-my-pm/` — Codex Skill for Oh My PM.

## Install

```bash
# From the oh-my-pm repository root:
bash installers/install-codex.sh
```

Or manually:

```bash
cp packs/codex/AGENTS.md /path/to/your/project/AGENTS.md
cp -r packs/codex/.agents/ /path/to/your/project/.agents/
```

## What it does

Once installed, Codex-compatible agents will operate with Oh My PM Head of Delivery behaviors and have access to the structured Oh My PM skill.
