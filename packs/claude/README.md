# Oh My PM — Claude Pack

Install Oh My PM for Claude Code.

## What this pack contains

- `CLAUDE.md` — Drop into your project root to activate Oh My PM for Claude Code.

## Install

```bash
# From the oh-my-pm repository root:
bash installers/install-claude.sh
```

Or manually:

```bash
cp packs/claude/CLAUDE.md /path/to/your/project/CLAUDE.md
```

## What it does

Once installed, Claude Code will operate as a Head of Delivery using Oh My PM. It will apply structured delivery thinking, produce actionable outputs, and support Persian/English bilingual workflows.
