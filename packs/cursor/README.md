# Oh My PM — Cursor Pack

Install Oh My PM for Cursor.

## What this pack contains

- `.cursor/rules/` — Cursor MDC rules for Oh My PM.

## Install

```bash
# From the oh-my-pm repository root:
bash installers/install-cursor.sh
```

Or manually:

```bash
cp -r packs/cursor/.cursor/rules/ /path/to/your/project/.cursor/rules/
```

## What it does

Once installed, Cursor will apply Oh My PM Head of Delivery behaviors via the MDC rules system. Rules activate based on task context.
