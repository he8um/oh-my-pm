# Oh My PM — Generic Agent Pack

Install Oh My PM for any AGENTS.md-compatible agent.

## What this pack contains

- `AGENTS.md` — Core Head of Delivery agent instructions in standard AGENTS.md format.

## Install

Copy `AGENTS.md` into your project root:

```bash
cp packs/generic/AGENTS.md /path/to/your/project/AGENTS.md
```

## What it does

Any agent that reads `AGENTS.md` will operate as a Head of Delivery using Oh My PM. This includes Codex, generic agent frameworks, and any tool that supports the `AGENTS.md` convention.
