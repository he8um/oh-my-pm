# Examples

Oh My PM includes synthetic examples to illustrate how a Head of Delivery agent handles real delivery scenarios.

All examples are fictional. No real companies or private data are used.

## Example projects

### Software project

`examples/software-project/`

An API integration project with delivery planning, risk assessment, and stakeholder update scenarios.

Available in English (`input.en.md`, `output.en.md`) and Persian (`input.fa.md`, `output.fa.md`).

### Marketing project

`examples/marketing-project/`

A product launch campaign with GTM planning and launch readiness scenarios.

Available in English and Persian.

### Product project

`examples/product-project/`

A PRD review and backlog prioritization scenario for a mobile app feature.

Available in English and Persian.

### Mixed delivery project

`examples/mixed-delivery-project/`

A cross-functional project with mixed English/Persian stakeholder context — illustrating bilingual delivery management.

## How to use examples

Examples show the expected input format and the kind of output Oh My PM produces.

Use them to:

- Understand what to ask the agent
- Review the output quality before trusting production outputs
- Calibrate bilingual quality
