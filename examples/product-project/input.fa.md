# مثال: پروژه محصول — ورودی (فارسی)

این یک مثال ساختگی است. هیچ شرکت واقعی یا داده خصوصی استفاده نشده است.

---

**ویژگی:** مرکز اعلان‌های درون‌برنامه‌ای
**تیم:** ۲ مهندس، ۱ طراح، ۱ PM
**تایم‌لاین:** ۵ هفته
**هدف:** کاهش تیکت‌های پشتیبانی از کاربرانی که اعلان‌های مهم حساب را از دست می‌دهند

**وضعیت PRD:** پیش‌نویس — نیاز به بررسی دارد

**بک‌لاگ فعلی (بدون اولویت‌بندی):**
۱. رابط کاربری صندوق ورودی اعلان‌ها
۲. زیرساخت push notification
۳. گزینه دایجست ایمیل
۴. ردیابی وضعیت خوانده/نخوانده
۵. تنظیمات ترجیحات اعلان
۶. پنل ادمین برای قالب‌های اعلان
۷. داشبورد آنالیتیکس برای تعامل اعلان
۸. پشتیبانی از deep link از اعلان‌ها
۹. گروه‌بندی اعلان بر اساس نوع
۱۰. محتوای اعلان چندزبانه
