# مثال: پروژه دلیوری ترکیبی — ورودی (فارسی)

این یک مثال ساختگی است که مدیریت دلیوری دوزبانه را نشان می‌دهد (زمینه کسب‌وکار فارسی، شناسه‌های فنی انگلیسی).

---

**پروژه:** طراحی مجدد فرآیند onboarding مشتری
**زمینه:** تیم از فارسی برای تصمیمات کسب‌وکار و انگلیسی برای کارهای فنی استفاده می‌کند.

**وضعیت فعلی:**
- بک‌لاگ تعریف شده اما اولویت‌بندی نشده
- تیم مهندسی آماده شروع است — در انتظار تأیید design
- بررسی استیک‌هولدرها هفته آینده برنامه‌ریزی شده
- قرارداد API بین frontend و backend نهایی نشده
- هیچ rollback plan وجود ندارد

**اهداف:**
- کاهش نرخ drop-off در فرآیند onboarding از ۴۵٪ به زیر ۲۵٪
- کاهش time-to-activate از ۸ دقیقه به زیر ۳ دقیقه

**تیم:** ارتباط ترکیبی فارسی/انگلیسی
