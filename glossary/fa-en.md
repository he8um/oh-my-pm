# Glossary — Persian / English

This glossary defines how Oh My PM handles PM and delivery terminology in bilingual contexts.

Technical identifiers (APIs, CLI, package names, config keys) always stay in English.

Management terms use established Persian transliterations where natural.

## Core PM terms

| English | Persian (preferred usage) | Notes |
| --- | --- | --- |
| Project | پروژه | Full Persian word |
| Delivery | دلیوری | Transliteration preferred in technical contexts |
| Product | پروداکت / محصول | Use محصول in formal, استیک‌هولدر-facing contexts |
| Scope | اسکوپ | Transliteration |
| Roadmap | رودمپ | Transliteration |
| Backlog | بک‌لاگ | Transliteration |
| Sprint | اسپرینت | Transliteration |
| Milestone | مایلستون | Transliteration |
| Release | ریلیز | Transliteration |
| Launch | لانچ | Transliteration |
| Stakeholder | استیک‌هولدر | Transliteration |
| Risk | ریسک | Transliteration |
| Blocker | بلاکر | Transliteration |
| Dependency | وابستگی | Full Persian word |
| Priority | اولویت | Full Persian word |
| Deadline | ددلاین / مهلت | Both acceptable; ددلاین in team context |
| Timeline | تایم‌لاین | Transliteration |
| Acceptance criteria | معیار پذیرش | Full Persian phrase |
| Status report | گزارش وضعیت | Full Persian phrase |
| Decision log | لاگ تصمیم / ثبت تصمیمات | Both acceptable |
| Retrospective | رترو / جلسه بازنگری | رترو in team context |
| Post-mortem | پست‌مورتم | Transliteration |
| Critical path | مسیر بحرانی | Full Persian phrase |
| Go/no-go | گو / نوگو | Transliteration |
| Agent handoff | هندآف ایجنت | Transliteration |

## Technical identifiers (always English)

Code, CLI commands, package names, file names, config keys, function names, and class names always stay in English — even in a Persian sentence.

Example:
```txt
اجرای دستور bash installers/install-claude.sh --dry-run را قبل از نصب واقعی انجام بده.
```
