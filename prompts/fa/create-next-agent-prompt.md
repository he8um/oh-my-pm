# پرامپت: ایجاد پرامپت برای جلسه بعدی ایجنت

از این پرامپت برای تولید یک پرامپت هندآف تمیز برای جلسه بعدی AI استفاده کنید.

---

```
تو به عنوان Head of Delivery با استفاده از Oh My PM عمل می‌کنی.

یک پرامپت هندآف مختصر برای جلسه بعدی ایجنت ایجاد کن. پرامپت باید:

۱. وضعیت فعلی پروژه را در ۳-۵ بولت پوینت خلاصه کند
۲. آنچه تصمیم گرفته شده را بیان کند (این موارد را دوباره بررسی نکن)
۳. آنچه باز است و نیاز به حل دارد را بیان کند
۴. اقدام بعدی مشخص را بیان کند — چه، توسط چه کسی، تا چه زمانی
۵. هر محدودیت حیاتی که ایجنت بعدی نباید نادیده بگیرد را شامل شود
۶. در مجموع زیر ۳۰۰ کلمه باشد

پدینگ نده. اطلاعاتی که ایجنت بعدی نیاز ندارد را شامل نشو.
زمینه موجود در مکالمه را تکرار نکن — آن را سنتز کن.

زمینه فعلی:
[وضعیت فعلی پروژه، تصمیمات گرفته‌شده، و قدم‌های بعدی را اینجا وارد کنید]
```
